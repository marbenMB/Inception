Get started with Inception

This project aims to broaden your knowledge of system administration by using Docker. You will virtualize several Docker images, creating them in your new personal virtual machine. This project consists in having you set up a small infrastructure composed of different services under specific rules. The whole project has to be done in a virtual machine. You have to use docker compose.

  • Execution :

    make

Services Each Per Container :

  • Wordpress + PHP-FPM
  • Nginx
  • Mariadb
  • Redis
  • Adminer
  • Static Website
  • Cadvisor
  • FTP
